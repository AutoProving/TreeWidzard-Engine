    For SEARCH:
        ./treewidzard -search [Flag Options] <search method> <property file>
    Flag Options:
        -pl       print loop
        -ps       print states
        -st       print state tree decomposition if a counterexample is found
    For PARSE:
        1- For PACE Format
            ./treewidzard -parse PACE <property file> <graph file> <tree decomposition file>
        2- For ABSTRACT Decomposition Verify
            ./treewidzard -parse ABSTRACT <property file> <abstract tree decomposition file>
